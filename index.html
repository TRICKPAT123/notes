Notely Map - By Naes

This system allows you to take notes and pin them on a map. I created these notes for traveling, so you can place notes on specific locations. You can also add an image to each note.

The system allows you to add notes or view the map; when you open the map, you can tap any location to add a note with a title and content, save it to My Notes on the home screen, edit it, and add an image, or you can also create a note directly from the New Note section without using the map.
